# Hero Mini RPG

A small C++ exercise demonstrating core OOP concepts: inheritance, composition, aggregation, and dependency through a simple RPG architecture.

## Concepts Demonstrated

- **Inheritance** — `Hero` extends `GameObject`
- **Composition** — `CoreStats` and `MysticalTattoo` live and die with the Hero
- **Aggregation** — `Guild` and `Weapon` exist independently of the Hero
- **Dependency** — `Potion` is consumed

## Compile & Run

```bash
g++ -std=c++14 -Wall main.cpp -o hero.exe
./hero.exe
```
